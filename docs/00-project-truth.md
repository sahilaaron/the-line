# 00 — Project Truth

The Line is a desktop-first interactive 3D experience for travelling through human history. One continuous world, not pages. The horizontal Line is the visual spine; navigation feels like changing position or depth inside that world.

## Feel

Spatial, cinematic but restrained, calm, premium, visually alive without noise. Reference: `Visual Reference.png` in repo root — deep space black, warm gold accents on the Line and active year, cool blue-white Earth, luminous theme orbs, era imagery living above the Line.

## Never

Generic SaaS layout, card grids, game HUDs, neon cyberpunk, decorative motion without meaning, fake data viz.

## Historical integrity

Never fabricate dates, quotes, inventors, sources, causation, or importance. All current content is placeholder and must stay clearly marked as such (see `src/data/`). Generated imagery is never presented as archival.

## Core concepts

- **Line View** — the main timeline world. A fixed temporal lens at screen centre; the Line moves beneath it.
- **Temporal Earth** — stylised procedural Earth above the active year, orbited by theme spheres.
- **Year on Line (YoL)** — a scene *inside* a year, reached by descending through the Earth's atmosphere.
- **Anchors** — the currently navigable years.
